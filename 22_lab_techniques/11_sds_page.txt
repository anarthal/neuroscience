SDS POLYACRYLAMIDE GEL ELECTROPHORESIS (SDS-PAGE): technique to separate proteins based on molecular weight
Polyacrylamide works better than agarose for proteins because it has smaller pores
Gel created by polymerizing acrylamide monomers
    Monomers form the meshwork, cross-linked by bis-acrylamide
    Monomers are neurotoxic
    Bigger bis-acrylamide concentrations => smaller pore size
    AMMONIUM PERSULFATE (APS): starts the polyacrylamide polymerization reaction
    TEMED: catalyzes the polyacrylamide polymerization reaction
We need to remove DNA first
Proteins need to be denatured before analysis
    Protein net charge depends on amino-acid composition and environment pH
        Even with the same molecular weight, the most negatively charged will move further
    Each protein has a != shape
        Even with the same molecular weight, globular proteins will move further because they adapt to pores better
    Use SDS to
        Denature proteins eliminates tertiary structure => eliminate the shape problem
        Cover all the protein with negative charges => eliminate the charge problem (gets masked)
    Proteins containing disulfide bonds are not denatured by SDS
        Introduce a reducing agent to remove the disulfide bond
        Beta-ME: cheap reducing agent
            Easy to handle, you can have it on the bench
            Aids detergents with solving proteins
            But degrades with oxygen and is volatile
        DTT: expensive
            More potent and quicker: with less concentration you can achieve the same effect
            Good when you can't dilute your sample a lot
            More expensive
            Less volatile and toxic than beta-ME
            Needs to be conserved at -20ºC
    But some very complex proteins are difficult to denature => heat at 95ºC
        If working with very hydrophobic proteins (e.g. transmembrane), milder heat, otherwise they aggregate
        Heating is used to help SDS. Once SDS denatures the protein, it no longer goes back to its original shape
In each well we insert the proteins and the sample buffer
    We need to quantify how much protein concentration we have because wells have a maximum
    Sample buffer: can be prepared beforehand
        SDS
        A dye: tracks progress when running the gel
        Glycerol: makes samples heavy
        pH buffer
    Reducing agents shouldn't be placed in the sample buffer because they're volatile, and preclude you from doing non-reducing gels
    Complete with distilled H2O => all wells should have the same volume
NON-REDUCING GEL: SDS-PAGE experiment that does not add a reducing agent. Checks if you have disulfide bonds
Components when running a gel
    Gel is run vertically. Proteins start at the top and move to the bottom
    Anode (-) is on top, cathode (+) is at the bottom
    RUNNING BUFFER: liquid phase present in reservoirs on top of the stacking gel and below the running gel
        Contains Tris-HCl and glycine
        Required to produce the stacking gel effect
    STACKING GEL: a low-concentration polyacrylamide gel (5%) placed on top of the chamber, so proteins settle on a plane
        Wells are located here
        When proteins are placed in the wells, they start at != heights. Keeping them like this yields incorrect results
        The stacking gel lets them pass until encountering the running gel => alignment in the vertical plane
        Neutral pH (6.8)
        pH difference with running gel + glycine pKA + chloride ions => stacking effect
    RUNNING GEL/RESOLVING GEL: higher-concentration polyacrylamide gel (~10%) placed at the bottom of the chamber, performing the actual separation
        Basic pH (8.8)
        Bigger proteins require lower acrylamide concentrations
        Gradient gels: top has big pores, bottom has small pores => higher dynamic range
Casting the gel
    Dissolve everything before adding the polymerization agents
    Pour stacking gel and wait for polymerization
        Add an isopropanol layer that gets removed later => smooth surface
    Pour running gel and wait for polymerization
        Place a comb to create the wells
    Remove the comb
    Add the running buffer
    Insert the sample (needs to happen after adding the running buffer so the running buffer doesn't disperse the sample)
The dye monitors where the proteins are running
    It runs faster than the proteins => when it exits the resolving buffer, it's time to stop the machine
Staining: required to visualize the proteins
    Happens within the gel
    General steps
        Wash to remove running buffer
        Fix to prevent protein bands from moving
        Wash to remove excess fixer
        Stain and incubate
        Wash to remove excess dye
    Coomassie: for higher concentrated proteins - the most used
    Silver: for lower concentration - mostly superseded by western blot
Blue-native gels: if you want to study the proteins without denaturing them first
    Useful to study complexes
    No SDS, no reducing agent, no heat
    E.g. 2D SDS-PAGE: separate 1st by complex, then by weight
